# Galaxy Simulator

這是一個使用Vite, Vue3和Pinia製作的3D銀河模擬器。在這個專案中，您可以創建自己的星球並在一個畫面中觀察它們。

## 專案介紹

這個專案的畫面分為上（85vh）和下（15vh）兩部分。

上方的主畫面是一個圓盤，裡面有三顆星球。當您點擊畫面的左右時，會透過圓盤的旋轉把左右的星球轉到正中間。這會透過角度和旋轉來讓星球看起來是立在圓盤上的3D感覺。

下方的畫面是一些功能：
- 功能1：新增星球
- 功能2：刪除星球

## 使用的技術
- Vite
- Vue 3
- TypeScript
- Pinia
- Tailwind + Postcss

## 如何使用

1. "clone" 此專案到您的本地機器。
2. "npm i" 安裝所有依賴項。
3. 開始創建您自己的銀河！
